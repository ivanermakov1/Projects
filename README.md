### 2. **20 Questions Game**
An AI-powered guessing game using Ollama.
- **Technologies**: Python, Ollama (Local AI)
- **Features**: Interactive gameplay, intelligent AI responses
- [View Project](./20-questions)

### 3. **Hangman Game**
Classic word-guessing game with random word selection.
- **Technologies**: C++
- **Features**: ASCII visualization, dictionary-based words, replay option
- [View Project](./hangman)

### 4. **Tic Tac Toe Game**
Strategic game against an intelligent computer opponent.
- **Technologies**: C++
- **Features**: AI blocking strategy, win/draw detection, input validation
- [View Project](./tictactoe)

Identifies Instagram accounts that don't follow you back.

- **Features**: Efficient set-based lookup, file I/O, fast algorithm
- [View Project](./insta-follower-checker)
